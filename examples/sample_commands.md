# Sample voice commands

The command set is configured in `ros2_ws/src/hri_voice_command/config/config.yaml`.

Example commands include:

- start system
- move to station 1
- screw the battery pack on the right
- screw the battery pack on the left
- return to the access point
- return to the waiting point
- go to the charging station
- have you finished the screwing operation
- stop system
